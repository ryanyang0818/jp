     ---
     遊戲規格

     核心玩法

     - 英文單字以隨機 X 位置從畫面頂端掉落
     - 同時最多 5 個單字在畫面中（隨關卡可增加）
     - 玩家在底部輸入框打字，按 Enter 送出
     - 比對所有正在掉落的單字，吻合者觸發爆炸並消除
     - 任一單字碰到底線 → 遊戲結束，顯示分數

     難度系統

     - 預設：速度隨時間遞增（每 15 秒加速一次）
     - 固定速度模式：右上角 checkbox 切換，勾起後維持初始速度不變

     計分

     - 消除一個單字 +1 分（可考慮字長加分）
     - 顯示目前分數與最高分（localStorage 儲存）

     ---
     技術架構

     渲染

     - <canvas> 負責繪製掉落單字、爆炸粒子效果
     - 背景：深色夜空（漸層），增加視覺對比

     爆炸效果

     - 消除時在單字位置生成 20~30 個彩色粒子
     - 粒子向外擴散並淡出（requestAnimationFrame 驅動）

     音效

     - 使用 Web Audio API 程式生成，不依賴外部音效檔
     - 爆炸音：短暫降頻的雜訊 burst（~0.2 秒）

     單字庫

     - 內建 200+ 常見英文單字，按長度分類
     - 初期生成較短單字（3-5 字母），加速後混入較長單字

     輸入區

     - 畫面底部固定輸入框，始終保持 focus
     - 打到一半的字母若有前綴吻合的單字，可高亮提示（optional 優化）

     實作步驟

     1. HTML 骨架：canvas + 底部輸入區 + 右上角控制列
     2. 遊戲狀態：words 陣列（含位置、速度）、particles 陣列、score、gameOver
     3. 主迴圈：requestAnimationFrame → 更新位置 → 繪製 → 碰地偵測
     4. 單字生成：setInterval 定時新增單字，頻率與速度依難度調整
     5. 輸入處理：Enter 事件 → 比對 → 消除 or 無效
     6. 爆炸系統：生成粒子 → 每幀更新位置與 alpha
     7. 音效：AudioContext 生成爆炸聲
     8. 遊戲結束：覆蓋層顯示分數 + Restart 按鈕
     9. LocalStorage：儲存最高分
